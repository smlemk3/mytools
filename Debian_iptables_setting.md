# Debian iptables 配置

安装iptables自启动
```Bash
apt update && apt install iptables-persistent ipset -y
```
ipset屏蔽列表 + 还原
```Bash
wget https://raw.githubusercontent.com/smlemk3/mytools/refs/heads/main/ipset.conf -O /etc/ipset.conf
ipset restore < /etc/ipset.conf
```
#########开机恢复ipset######
```Bash
cat > /etc/network/if-pre-up.d/ipset << "EOF"
#!/bin/sh
ipset restore < /etc/ipset.conf
#iptables-restore < /etc/iptables/rules.v4
systemctl restart netfilter-persistent
exit 0
EOF

chmod +x /etc/network/if-pre-up.d/ipset
```
# 开启端口转发
```Bash
echo net.ipv4.ip_forward=1 >> /etc/sysctl.conf
echo net.ipv6.conf.all.forwarding=1 >> /etc/sysctl.conf
```
iptables-persistent 的配置文件位置 /etc/iptables/rules.v4 /etc/iptables/rules.v6
Ipv4 防火墙规则
```Bash
cat > /etc/iptables/rules.v4 << "EOF"
*filter
:INPUT ACCEPT [0:0]
:FORWARD ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
-A INPUT -m set --match-set blacklist src -j DROP
-A FORWARD -m set --match-set blacklist src -j DROP
-A INPUT -i lo -j ACCEPT
-A INPUT -d 127.0.0.0/8 ! -i lo -j REJECT --reject-with icmp-port-unreachable
-A INPUT -m state --state RELATED,ESTABLISHED -j ACCEPT
-A INPUT -p tcp -m tcp --dport 80 -j ACCEPT
-A INPUT -p tcp -m tcp --dport 3306 -j DROP
-A INPUT -p tcp -m state --state NEW -m tcp --dport 22 -j ACCEPT
-A INPUT -p tcp -m state --state NEW -m tcp --dport 443 -j ACCEPT
-A INPUT -p tcp -m state --state NEW -m tcp --dport 52222 -j ACCEPT
-A INPUT -p icmp -m icmp --icmp-type 8 -j ACCEPT
-A INPUT -j REJECT --reject-with icmp-port-unreachable
# 防爆破
-A FORWARD -p tcp -m tcp --dport 59999 -m state --state NEW -m recent --update --seconds 60 --hitcount 5 --name RDP1_BOMB --mask 255.255.255.255 --rsource -j DROP
-A FORWARD -p tcp -m tcp --dport 59999 -m state --state NEW -m recent --set --name RDP1_BOMB --mask 255.255.255.255 --rsource -j ACCEPT
-A FORWARD -p tcp -m tcp --dport 39999 -m state --state NEW -m recent --update --seconds 60 --hitcount 5 --name RDP2_BOMB --mask 255.255.255.255 --rsource -j DROP
-A FORWARD -p tcp -m tcp --dport 39999 -m state --state NEW -m recent --set --name RDP2_BOMB --mask 255.255.255.255 --rsource -j ACCEPT
-A OUTPUT -j ACCEPT
COMMIT
# Completed on Mon Nov  3 05:07:24 2025
*nat
:PREROUTING ACCEPT [0:0]
:INPUT ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
:POSTROUTING ACCEPT [0:0]
#端口转发
#-A PREROUTING -p tcp -m tcp --dport 38888 -j DNAT --to-destination 66.66.66.66:59999
#-A PREROUTING -p tcp -m tcp --dport 39999 -j DNAT --to-destination 66.66.66.66:39999
#-A POSTROUTING -d 66.66.66.66/32 -p tcp -m tcp --dport 59999 -j MASQUERADE
#-A POSTROUTING -d 66.66.66.66/32 -p tcp -m tcp --dport 39999 -j MASQUERADE
COMMIT
# Completed on Mon Nov  3 05:07:24 2025
EOF

iptables-restore < /etc/iptables/rules.v4
```
IPv6 防火墙规则
```Bash
cat <<EOF > /etc/iptables/rules.v6
# Generated by ip6tables-save v1.8.9 (nf_tables) on Sat May 11 17:44:56 2024
*filter
:INPUT ACCEPT [0:0]
:FORWARD ACCEPT [0:0]
:OUTPUT ACCEPT [0:0]
-A INPUT -i lo -j ACCEPT
-A INPUT -p tcp -m tcp --dport 3306 -j DROP
-A INPUT -s 2602:80d:1000:b0cc:e::/80 -j DROP
-A INPUT -s 2620:96:e000:b0cc:e::/80 -j DROP
-A INPUT -s 2602:80d:1003::/112 -j DROP
-A INPUT -s 2602:80d:1004::/112 -j DROP
COMMIT
# Completed on Sat May 11 17:44:56 2024
EOF

iptables6-restore < /etc/iptables/rules.v4
```
